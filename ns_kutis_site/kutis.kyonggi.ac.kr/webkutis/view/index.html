로그인 영역

  • 제적생은 로그인이 불가능 합니다.
  • 비밀번호 5회 입력 오류시 본인인증으로 비밀번호 초기화 후 사용하시기 바랍니다.
  • [동시로그인 금지] 같은 아이디로 로그인 한 경우 이전 로그인은 강제 로그아웃됩니다.
  • 단일사용승인(SSO) 서비스는 한 번 로그인으로 홈페이지, 이메일 등을 추가 정보 입력없이 로그인 할 수 있는 기능입니다.
  • 단일사용승인(SSO) 서비스에 의해 로그아웃이 되었습니다. 라는 메세지가 띄는 사용자는 위 체크박스를 해제하지 말고 사용하시기 바랍니다.
  • 등록금(교육비납입증명서)출력
  • 본인인증으로 개인정보(학번/사번)찾기
  • 본인인증으로 비밀번호 초기화
등록금고지서(교육비납입증명서)출력
클릭 -> 학부모용 간편서비스 출력바로가기
  • 비밀번호를 모르는 경우

    - 학생 학번/생년월일로 로그인하여 출력바로가기 화면으로 이동

  • 출력절차 간소화하여 보다 빠르고 편리한 서비스 제공
  • 교육비납입증명서는 등록금납부 영수증으로 사용가능
